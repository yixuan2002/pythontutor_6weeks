📖 第一部分

概念講解

for 迴圈、while 迴圈，加上 break 與 continue 兩個控制小幫手。

3-1　for 迴圈 + range()

for 迴圈用來「重複做固定次數的事」，最常搭配 range() 使用。把 for i in range(5) 唸成：「對 0 到 4 的每一個數字 i，做以下的事。」

💡
為什麼程式都從 0 開始數？

想像你住在巷子的第一間房，距離巷口是 0 步，不是 1 步。電腦排資料也一樣，第一個位置「距離起點 0 格」，所以編號從 0 開始。

口訣：range(5) 給你 0 到 4，共 5 個數字；最後一個永遠是 n−1。

3-2　while 迴圈

while 迴圈用來「在某個條件成立時，一直重複執行」，次數不固定 —— 很適合做遊戲的主迴圈。

⚠️
無限迴圈是新手最常踩的雷

如果 while 的條件永遠是 True，程式會一直跑停不下來。要確保迴圈裡有東西會讓條件變成 False（例如 count += 1）。

真的卡住了？在終端機按 Ctrl + C 可以強制停止。

3-3　break 與 continue

這兩個關鍵字讓你更精準地控制迴圈。

break
立刻結束整個迴圈，像打電動按「離開遊戲」
continue
跳過這一圈，直接進下一圈

3-4　for vs while：什麼時候用哪個？

for — 次數固定
  • 重複做 10 次某件事
  • 印出 1 到 100 的數字
  • 對清單裡每個元素做事
while — 條件控制
  • 等使用者輸入正確答案才停
  • 遊戲持續進行直到 Game Over
  • 倒數計時到 0
