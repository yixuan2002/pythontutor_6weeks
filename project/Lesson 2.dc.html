📖 第一部分

概念講解

if / elif / else 的語法，加上比較運算子與邏輯運算子。

2-1　if / elif / else 基本結構

條件判斷讓程式能「做選擇」。Python 從上往下檢查每個條件，一旦有一個成立，就執行它底下的區塊，其他都跳過。

🔑
縮排是 Python 的靈魂

Python 用「縮排」（4 個空白或 1 個 Tab）來標示哪些程式碼屬於同一個區塊。縮排錯了，程式會直接跑不起來（IndentationError）。

其他語言用 { } 括號，Python 用縮排取代括號。

2-2　比較運算子

比較運算子會回傳一個布林值（True 或 False），正好可以拿來當 if 的條件。

運算子唸法範例結果
{{ c.sym }}{{ c.name }}{{ c.ex }}{{ c.res }}
⚠️
最常見的錯誤：= 和 == 不一樣！

= 是「指定」：把右邊的值放進左邊的變數。

== 是「比較」：問左右兩邊是否相等，回傳 True / False。

寫 if score = 100: 會直接 SyntaxError，要寫 if score == 100:

2-3　邏輯運算子：and / or / not

想一次判斷多個條件時，用邏輯運算子把它們組合起來。

and
兩個條件都要成立，整句才成立
or
只要有一個成立，整句就成立
not
把 True 變 False、False 變 True
